Niqqud-er

הוראות שימוש

  • הדבק טקסט ולחץ Submit (ברירת מחדל: בלי ניקוד).
  • Hover על מילה: מציג ניקוד ומשאיר.
  • קליק על מילה: מחליף ניקוד/ללא ניקוד ומעדכן מיקום נוכחי.
  • ← מגלה מילה הבאה, → מחזיר מילה קודמת; המילה הפעילה מודגשת קצרות.

How To Use

  • Paste text with niqqud, then click Submit. The text will be converted to remove niqqud.
  • Hover a word: reveal niqqud and keep it.
  • Click a word: toggle niqqud on/off and set current position.
  • Use left/right arrows: left reveals next word, right hides previous.

If word count matches the original, this text is used as-is.
